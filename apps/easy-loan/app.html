Easy Loan

Are you planning to take out a loan but want to know how much you will be paying back in total? Here is a loan calculator app that can help you! Simply enter the amount you want to borrow, the interest rate, and the duration of the loan, and thr app will calculate your monthly payment and the total amount you will pay back over the life of the loan.

Give it a try!

Easy Loan: Amortization calculator

Whether you're a borrower looking to understand your loan payments or a financial professional looking for a tool to help your clients, the Amortization Calculator is a useful and easy-to-use tool that can help you make informed loan decisions.

Instructions:

  1. Enter Loan Amount.
  2. Enter the number of payments.
  3. Enter the interest rate.
  4. Click the button to calculate the amortization schedule.

/*
* Description
* A helper function that I use to shorten targeting a DOM element
* 1. Takes in the type and name of an element and returns it into a variable
* eg: let fizzValue = getElementBy('id', "fizzValue").value;
*/
function getElementBy(type, value)
{
    if(type === "id" || type === "id")
        return document.getElementById(value);
    
    if(type === 'class' || type === 'Class' || type === 'ClassName' || type === 'className')
        return document.getElementsByClassName(value);
    
    if(type === 'tag' || type === 'Tag' || type === 'TagName' || type === 'tagName')
        return document.getElementsByTagName(value);
}
